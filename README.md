# Siamese Network Training and Face Recognition
This repository contains notebooks and resources to get started with face recognition
Steps : 
1. clone the repository 
2. Create two folders, one named "people" and other as "test".
3. "people" folder should contain the photos of people who are to be considered for face recognition (e.g. your family members, friends). One photo each. The photograph should strictly be named as the name of the person in the photo or example - Abhishek.jpg
4. "test" folder should contain  the images on which you want to test the face recognition algorithm.
